¿De dónde saco el CSV? Guía de 2 minutos

Importante: el export normal de Search Console no vale — te da consultas y páginas por separado, sin cruzar. Necesitas una fila por cada par keyword + URL. Tres formas gratis:

1 · Search Analytics for Sheets (recomendada, 2 min): instala el complemento gratuito en Google Sheets → Extensiones → Search Analytics for Sheets → Open Sidebar → elige tu propiedad → en Dates elige All time → en Group By selecciona Query y Page → Request Data → Archivo → Descargar → CSV.

2 · Looker Studio (5 min): informe nuevo → fuente Search Console → tabla "Impresión de URL" (la única que cruza consulta con página) → tabla con dimensiones Consulta + Landing Page y métricas Clics, Impresiones y Posición → tres puntos → Exportar CSV.

3 · Semrush / Ahrefs: export de posiciones orgánicas con URLs. Funciona, pero con datos estimados; Search Console es la verdad.

La herramienta te da el vistazo; la decisión es tuya. Cada proyecto es un mundo: que una keyword tenga varias URLs puede ser canibalización real o convivencia buscada (categoría + guía, términos de marca, SERPs con intención mixta…). Antes de fusionar, redirigir o desindexar nada, revisa la intención de búsqueda, la SERP real, la estacionalidad y el papel de cada URL en el negocio. Esto te señala dónde mirar; interpretarlo y decidir es el trabajo del SEO.
